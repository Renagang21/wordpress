Advanced membership management plugin for WordPress.

== Description ==
Rena Members is a powerful membership plugin for WordPress, providing flexible user management, content restriction, and user profile features.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/rena-members` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings -> Rena Members screen to configure the plugin.

== Frequently Asked Questions ==
= Can I restrict content to specific user roles? =
Yes, you can restrict any post or page to specific user roles.

= Does this plugin support custom fields for user profiles? =
Yes, you can add custom fields to registration and profile forms.
